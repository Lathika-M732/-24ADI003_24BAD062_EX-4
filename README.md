SCENARIO 1 – MULTINOMIAL NAÏVE BAYES

Dataset: https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

Multinomial Naïve Bayes is used for text classification to identify SMS messages as Spam or Ham. The text data is cleaned and converted into numerical form using Count Vectorization / TF-IDF. The dataset is split into training and testing sets, the model is trained, and performance is evaluated using accuracy, precision, recall, F1-score, and confusion matrix.


SCENARIO 2 – GAUSSIAN NAÏVE BAYES

Gaussian Naïve Bayes is used to classify flower species based on features like sepal length, sepal width, petal length, and petal width. After preprocessing and feature scaling, the dataset is split into training and testing sets. The model is trained and evaluated using accuracy, precision, recall, F1-score, and confusion matrix, with visualization of decision boundaries and class probabilities.
